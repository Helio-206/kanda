# Roadmap

The GitHub issues are organized into four milestones. Each milestone has one epic issue and several task issues.

## Milestone 1 - MVP Core

Goal: consolidate the product foundation.

Epic:

- [Epic: MVP Core](https://github.com/Helio-206/kanda/issues/20)

Key tasks:

- web/mobile architecture split;
- central occurrence model;
- KANDA Score;
- public tracking by code;
- community confirmations;
- duplicate detection;
- entity dashboard foundation;
- CI and contribution workflow.

## Milestone 2 - AI Layer

Goal: build the automated analysis and API contract layer.

Epic:

- [Epic: AI Layer](https://github.com/Helio-206/kanda/issues/21)

Key tasks:

- simulated analysis engine;
- entity and service-area matching;
- editable suggested tags;
- FastAPI backend;
- OpenAPI contract.

## Milestone 3 - Maps

Goal: make the territorial view operational.

Epic:

- [Epic: Maps](https://github.com/Helio-206/kanda/issues/22)

Key tasks:

- occurrence map with real map tiles;
- risk-based markers;
- geospatial database structure;
- nearby queries and future heatmap readiness.

## Milestone 4 - Demo

Goal: prepare a reliable end-to-end commercial demonstration.

Epic:

- [Epic: Demo](https://github.com/Helio-206/kanda/issues/23)

Key tasks:

- citizen mobile app;
- web/mobile/backend integration;
- presentation-safe demo data and fallback flow.
