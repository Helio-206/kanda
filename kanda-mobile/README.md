# Kanda Mobile

Kanda é uma app mobile comunitária para Angola, feita com React Native e Expo.

## Arranque

```bash
npm install
npm run start
```

## MVP

- Login simples por telefone
- Painel inicial com resumo da comunidade
- Reporte de ocorrência
- Mapa rápido das ocorrências
- Perfil simples do utilizador
